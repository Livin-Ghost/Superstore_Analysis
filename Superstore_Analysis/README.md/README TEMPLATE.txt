# 📊 Retail Sales Analysis Project

## 🚀 Project Highlights
- Cleaned and transformed raw sales data
- Performed exploratory data analysis (EDA)
- Generated business insights for decision-making
- Built visualizations to identify trends and patterns

## 🔍 Overview
This project analyzes retail sales data to uncover insights into product performance, customer behavior, and regional trends.

## 🎯 Objectives
- Identify total revenue
- Find top-selling products
- Analyze monthly sales trends
- Identify top customers
- Compare regional performance

## 🛠 Tools Used
- Python (Pandas, Matplotlib)
- Jupyter Notebook

## 📈 Key Insights
- A small number of products generate the majority of revenue
- Sales show strong monthly trends (seasonality)
- Top customers contribute significantly to total sales
- Certain regions outperform others consistently

## 📊 Visualizations
- Top 10 products by sales
- Top 5 categories
- Monthly sales trend

## ▶️ How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Open Jupyter Notebook:
   jupyter notebook
4. Run all cells
